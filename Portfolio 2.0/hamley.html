WEBSITE REDESIGN



Overview :



Hamley’s is the world's oldest toy retailer . While going through its website I felt there was a room for improvement in the UI so I took up the challenge and redesigned it to make it look more organized and user friendly




Approach







Research and Understanding


The website followed a theme but there is inconsistency overall. The website needed to be de cluttered for the user to navigate through the website easily.



Competitive analysis:


    1.Firstcry.com

i. The target audience of the website is similar to Hamley’s.
ii. The UI for the website is better but it is still distracting all the areas demand for equal amount of attention.
iii. There are too many items on the navigation bar.

    2. Funcorp.com

i. The target audience of the website is similar to Hamley’s.
ii. The UI is cluttered as all the different brands having different colors are displayed on the center of the front page which makes it look cluttered.
iii. The UX of the website is similar to Hamley’s website.

User Flow:



After Conducting user interviews and analyzing them i found that there is no major issues with the UX of the website except there was no rating section available


Almost all the users found the UI very cluttered and uneasy to browse through






Design principles not followed:


i. Hierarchy
ii. Consistency
iii. Accessibility


Outcome of research:


i. Overwhelming home page
ii. Many tabs on Navigation bar
iii. Inconsistent fonts
iv. Design principles not followed



Goal



i. To declutter the home page
ii. To organize the navigation bar
iii. Make the buttons uniform across pages


Ideation


My major Goal was to make the text legible but to keep the essence of the website same so i stuck with the original color theme of the website. Before designing i created a mood board for the website.

1.Before
